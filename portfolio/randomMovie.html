random movie

Un projet personnel sur lequel je travaille toujours conçu dans le but de me familiariser davantage avec Javascript et le fonctionnement des API. Sur ce site Web, l'utilisateur pourrait entrer le type de film ou émission télévisée qui l'intéresse, après quoi il recevra une recommendation au hasard en plus de quelques informations supplémentaires.

Contrairement aux travaux précédents, j'ai utilisé Figma pour concevoir l'apparence du site Web sous différentes résolutions ; pour ordinateurs de bureau, tablettes et mobiles. Le site Web lui-même reste un travail en cours pour le moment.

Ce projet prévoit d'utiliser l'API de TheMovieDB.

outils

Figma / HTML5, CSS3, SASS, Javascript
